Portfolio Overview

I enhanced a single artifact across three key categories required for the CS 499 capstone: software design and engineering, algorithms and data structures, and databases.   • Databases: Added IndexedDB persistence with schema design, a unique SKU index, and transaction-based CRUD operations.
